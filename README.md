## Page Optimizer — Модуль оптимизации страниц для HostCMS 7

Модуль автоматической оптимизации HTML-страниц (минификация, объединение CSS/JS, lazy-load изображений, DNS Prefetch, Preconnect, Critical CSS и др.).

## Возможности

- Минификация HTML
- Объединение и минификация CSS/JS
- Lazy Loading изображений + конвертация в WebP/AVIF
- DNS Prefetch / Preconnect / Preload шрифтов
- Вставка Critical CSS
- Сбор статистики экономии

## Установка

1. Скопируйте папку `modules/page_optimizer` в `/modules/`
2. Скопируйте папку `admin/page_optimizer` в `/admin/`
3. Активируйте модуль в разделе **Модули**
