# hostcms-modules

Production-oriented additional modules for HostCMS 7.x.

## Modules

- `uvlist` — neutral List-like module for named lists and nested list items.

## Current focus

The first milestone is a working native `uvlist` module built against HostCMS 7 Admin_Form and ORM conventions. The old List module is used only as a functional specification, not as code to be blindly ported.
